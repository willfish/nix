# Daily Notes

Use this for daily notes in `~/Notes/YYYY-MM-DD/today.md`.

Default workflow:
1. Read yesterday's note to see planned work and PR references.
2. Summarise completed work in "What I did yesterday".
3. Leave "What I plan to do today" blank unless the user gives tasks.
4. Keep entries concise: one line per item, no sub-bullets.

Read `references/daily-notes.md` before editing notes.
